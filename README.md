# Spring Boot Spring State Machine Example
This is Spring Boot Spring State Machine Example

## Prerequisites

* Java 11
* Maven 3.3+


## Running the Project

```sh
mvn spring-boot:run
```


## Used Technologies

* Spring Boot 2.4.0
* Spring Statemachine Core
* H2
* Spring Boot Web
* Spring Boot Jpa
* Spring Boot Actuator
* Lombok
* Dev Tools
